# 근로시간 · 수당 규정 정리 (Copilot / 코드 구현용)

> 이 문서는 **알바/근로 스케줄 관리 서비스**에서 근로시간·수당 계산 로직을 구현하기 위한 기준 정리 문서입니다.
> GitHub Copilot이 로직을 추론하기 쉽도록 **조건 / 계산식 / 적용 대상**을 명확히 분리했습니다.
> 확정된 workShift 기준으로 급여를 계산하고, 실시간 날짜를 체크하여, 예상임금 등을 함께 출력할 수 있게한다.
> 하단의 패키지구조는 단순히 예시일 뿐이며(ChatGPT) 고려하지않아도된다.
---

## 1. 기본 개념

* **통상임금**: 일반적으로 `시급`을 의미
* **5인 미만 사업장**: 연장·야간·휴일근로 가산수당 **미적용**
* **5인 이상 사업장**: 근로기준법상 모든 가산수당 적용

---

## 2. 휴게시간 (Break Time)

### 적용 대상

* 전체 사업장

### 조건

* 1일 **4시간 이상** 근무 → **30분 이상** 휴게
* 1일 **8시간 이상** 근무 → **1시간 이상** 휴게

### 의무

* 근무 **도중** 제공해야 함 (근무 전/후 제공 불가)

```text
if (dailyWorkTime >= 8h) breakTime >= 1h
else if (dailyWorkTime >= 4h) breakTime >= 30m
```

---

## 3. 주휴일 · 주휴수당

### 적용 대상

* 전체 사업장

### 조건

* 1주 **15시간 이상** 근무
* 해당 주 **결근 없음**
* 근로계약서 상, 1년 이상 계약이라는 문구가 명시되어야함. 해당 여부 판단

### 내용

* **1일 유급휴일** 제공
* 주휴수당 지급

### 계산식

```text
주휴수당 = (주 근로시간 ÷ 40) × 8 × 시급
```

### 비고

* 결근 시 해당 주 **주휴수당 미지급**

---

## 4. 연장근로수당 (Overtime Pay)

### 적용 대상

* **5인 이상 사업장만**

### 조건

* 1일 8시간 초과 **또는**
* 1주 40시간 초과

### 내용

* 통상임금의 **50% 가산**

### 계산식

```text
연장수당 = 초과근로시간 × 시급 × 1.5
```

### 한도

* **주 최대 12시간**

---

## 5. 야간근로수당 (Night Pay)

### 적용 대상

* **5인 이상 사업장만**

### 조건

* **22:00 ~ 06:00** 근무

### 내용

* 통상임금의 **50% 가산**

### 계산식

```text
야간수당 = 야간근로시간 × 시급 × 1.5
```

---

## 6. 휴일근로수당 (Holiday Pay)

### 적용 대상

* **5인 이상 사업장만**

### 조건

* 유급휴일(주휴일, 공휴일)에 근무

### 내용

* 8시간 이내 → **50% 가산**
* 8시간 초과 → **100% 가산**

### 계산식

```text
휴일근로수당 = 근로시간 × 시급 × (1.5 or 2.0)
```

---

## 7. 중복 가산 규칙

### 연장 + 야간 근로

* 야간시간(22~06)에 연장근무 발생 시

```text
시급 × 2.0 (총 100% 가산)
```

### 휴일 + 야간 근로

* 휴일근로 가산 + 야간 가산 **중복 적용**

---

## 8. 법정 근로시간

### 적용 대상

* **5인 이상 사업장만**

```text
1일 8시간
1주 40시간
```

* 초과 시 연장근로로 분류

---

## 9. 연장근로 한도

* **주 최대 12시간**
* 초과 시 **근로기준법 위반**

## 13. Copilot 친화적 설계 팁

* `Policy` 단위로 수당 분리 → 테스트/확장 쉬움
* **boolean isFiveOrMoreEmployees** 명시적으로 전달
* 시간 계산은 `LocalDateTime` + `Duration` 기반 권장
* 중복 가산은 **합산 방식**이 아닌 **최종 배율 방식** 추천
